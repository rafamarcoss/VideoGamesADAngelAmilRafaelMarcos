# VideoGamesADAngelAmilRafaelMarcos
Este proyecto implementa una solución avanzada para la gestión de catálogos de videojuegos, fundamentada en una arquitectura de persistencia políglota. El sistema utiliza Java 17 y Maven para coordinar dos motores de base de datos con roles especializados: SQL (H2) y NoSQL (MongoDB), garantizando eficiencia y escalabilidad.
